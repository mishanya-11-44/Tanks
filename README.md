# танки
Классическая аркадная игра "Танчики", написанная на Python с использованием библиотеки Tkinter. Игрок управляет танком, сражается с вражескими танками (ботами), собирает бонусы и разрушает окружение.

## 🎮 Игровой процесс

- Управление танком осуществляется с помощью клавиш **W, A, S, D**.
- Выстрел производится клавишей **ПРОБЕЛ**.
- Камера автоматически следует за танком игрока.
- Цель — уничтожить всех вражеских танков, собирая бонусы для улучшения характеристик.

## 🧱 Типы блоков

| Символ | Описание          | Взаимодействие                          |
|--------|-------------------|------------------------------------------|
| `g`    | Земля (GROUND)    | Проходимая поверхность                  |
| `w`    | Вода (WATER)      | Замедляет движение танка                |
| `c`    | Бетон (CONCRETE)  | Непроницаем для пуль                    |
| `b`    | Кирпич (BRICK)    | Разрушается одним выстрелом             |
| `m`    | Бонус "Боезапас"  | Добавляет +10 к патронам                |
| `n`    | Бонус "Скорость"  | Удваивает скорость танка                |
| `f`    | Бонус "Топливо"   | Восстанавливает 1000 единиц топлива     |

## 🚀 Особенности

- **Динамическая камера**: плавно следует за игроком, ограничена границами карты.
- **Боты с ИИ**: вражеские танки самостоятельно перемещаются, находят цель и стреляют.
- **Физика столкновений**: танки не могут проходить сквозь стены и друг друга.
- **Бонусная система**: подбор предметов улучшает боевые характеристики.
- **Разрушаемое окружение**: кирпичные стены можно пробивать снарядами.

## 🛠️ Управление

| Действие        | Клавиша     |
|----------------|-------------|
| Движение вверх | `W`         |
| Движение вниз  | `S`         |
| Движение влево | `A`         |
| Движение вправо| `D`         |
| Выстрел        | `ПРОБЕЛ`    |
| Ручное перемещение камеры | `↑` `↓` `←` `→` |

## 📁 Структура проекта
.
├── main.py # Главный файл, запуск игры и обработка событий
├── world.py # Модуль мира: карта, камера, блоки, физика окружения
├── units.py # Классы юнитов: танк (игрок/бот) и ракета
├── hitbox.py # Класс Hitbox для обработки столкновений
├── missiles_collection.py # Управление всеми ракетами на поле
├── tanks_collection.py # Управление всеми танками (игрок + боты)
├── texture.py # Загрузка и хранение спрайтов
└── img/ # Папка с изображениями (.png)

## ▶️ Запуск игры

Убедитесь, что у вас установлен Python 3.7+ и все необходимые файлы изображений лежат в папке `img` относительно корня проекта.

```bash
python main.py
🖼️ Необходимые ресурсы
В папке img/ должны присутствовать следующие файлы (примерные названия, сверьте с кодом):

tank_up.png, tank_down.png, tank_left.png, tank_right.png — текстуры вражеских танков.

tank_up_player.png, tank_down_player.png, tank_left_player.png, tank_right_player.png — текстуры танка игрока.

tank_destroy.png — анимация уничтожения.

brick.png, wall.png, water.png — текстуры блоков.

bonus.png, bonus2.png, bonus3.png — бонусы.

missile_up.png, missile_down.png, missile_left.png, missile_right.png — снаряды.

здоровье1.png … здоровье5.png — индикаторы здоровья (опционально).

🧠 Искусственный интеллект ботов
Боты периодически меняют направление движения.

При обнаружении игрока выравниваются по оси и открывают огонь.

Если бот сталкивается с другим танком или стеной, он выбирает новый маршрут.

📦 Зависимости
Проект использует только встроенные модули Python:

tkinter — для графического интерфейса.

random — для случайных генераций и ИИ.

📄 Лицензия
Проект создан в образовательных целях. Свободно используется и модифицируется.
